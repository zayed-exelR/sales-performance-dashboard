Sales Performance Dashboard (SQL + Power BI)

## Project Overview
This project analyzes sales performance using SQL and Power BI.  
The dataset contains 10,000+ sales transactions from 2021–2023.  
The goal was to identify key business insights and build an interactive dashboard.

---

## Tools Used
- MySQL
- SQL
- Power BI
- DAX

---

## Key KPIs
- Total Revenue: 23.47M
- Total Profit: 4.10M
- Profit Margin: 17.47%
- Total Orders: 10,000
- Average Order Value: 2,347

---

## Analysis Performed
- Monthly revenue trend analysis
- Region-wise sales performance
- Category revenue and profit comparison
- Top 10 high-value customers
- Profit margin analysis

---

## Dashboard Features
- Interactive KPI cards
- Monthly revenue trend visualization
- Region performance comparison
- Category-wise revenue and profit analysis
- Top customer identification
- Year and region slicers for filtering

---

## Business Insights
- Revenue distribution is balanced across regions.
- North region generated the highest revenue.
- Customer revenue distribution shows stable demand across the customer base.
- Monthly revenue trend remained consistent across 2021–2023.
